# FastAPI Practice

A basic social media API built using FastAPI for learning FastAPI, Pytest, and Pydantic.

## Routes
- Get all posts.
- Get single post.
- Get posts with comments.
- Add post.
- Add comments.
- Get comments on a post.
- Create new user
